# Dynamic Programming

- 입력의 크기가 가장 작은 부분 문제부터 해를 구하여 테이블에 저장해 놓고 이를 이를 이용해서 입력 크기가 보다 큰 문제의 해를 점진적으로 만들어 가는 상향식 접근 방법
- 각각의 작은 문제는 원래 문제와 동일, 입력의 크기만 작음
- 작은 문제들은 서로 독립일 필요가 없음
