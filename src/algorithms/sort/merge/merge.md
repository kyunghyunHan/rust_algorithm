## Merge Sort
 - 주어진 배열을 동일한 크기의 2개의 부분배열로 분할하고, 각 부분배열을 순환적으로 합병 정렬로 정렬한 후, 정렬된 두 부분배열을 결합하여 하나의 정렬된 배열을 만드는 방식의 알고리즘
  
## 성능과 특징
- 안정적인 정렬 : 합병 과정에서 동일한 두 데이터에 대해서 항상 왼쪽 데이터를 먼저 선택함
- 제자리 정렬 알고리즘이 아님
#### )Recurrence Equation
- T(n) = ⎰ Θ(1),           n = 1 ⎱
         ⎱ 2T(n/2) + Θ(n), n ≥ 2 ⎰
#### )Closed Type

- T(n) = Θ(n log n)
